## 잔액 조회
```mermaid
sequenceDiagram
  actor Client as 사용자
  participant API as API
  participant Service as 잔액 서비스
  participant DB as DB
  activate Client
  Client ->>+ API: 잔액 조회 요청
  API ->>+ Service: 잔액 조회 처리
  Service ->>+ DB: 금액 테이블 조회
  DB -->>- Service: 잔액 정보 반환
  Service -->>- API: 잔액 조회 결과
  API -->>- Client: 결과 전달
  deactivate Client
```

## 잔액 충전
```mermaid
sequenceDiagram
actor Client as 사용자
participant API as API
participant Service as 잔액 서비스
participant DB as DB

activate Client
Client ->>+ API: 잔액 충전 요청
API ->>+ Service: 잔액 충전 처리
activate Service
Service ->>+ DB: 잔액 테이블 조회
activate DB
DB -->> Service: 사용자 잔액 조회 결과
deactivate DB
Service ->> DB: 잔액 테이블 잔액 증가
activate DB
opt 최대 잔액 초과
DB -->> Service:  잔액 증가 실패
end
DB -->> Service: 잔액 증가 결과
deactivate DB
Service ->> DB: 잔액 증가 이력 생성
activate DB
DB -->> Service: 이력 생성 결과
deactivate DB
Service -->> API: 잔액 충전 결과
deactivate Service
API -->>- Client: 결과 전달
deactivate Client
```

## 선착순 쿠폰
```mermaid
sequenceDiagram
        actor 사용자
        participant API
        participant 쿠폰 서비스
        participant DB

activate  사용자
사용자 ->> API: 쿠폰 발급 요청
API ->> 쿠폰 서비스: 쿠폰 발급 처리
activate 쿠폰 서비스 
쿠폰 서비스 ->> DB: 쿠폰 잔여 수량 조회
activate DB
DB -->> 쿠폰 서비스: 쿠폰 잔여 수량 반환
deactivate DB
opt 잔여 수량 <= 0
쿠폰 서비스 -->> API: 쿠폰 발급 실패
end
쿠폰 서비스 ->> DB: 쿠폰 잔여 수량 감소
activate DB
DB -->> 쿠폰 서비스: 쿠폰 잔여 수량 감소 결과
deactivate DB
쿠폰 서비스 ->> DB: 사용자 테이블에 쿠폰 저장
activate DB
DB -->> 쿠폰 서비스: 쿠폰 저장 결과
deactivate DB
쿠폰 서비스 -->> API: 쿠폰 발급 성공
API -->> 사용자: 결과 전달
deactivate 쿠폰 서비스 
deactivate 사용자
```

## 주문/결제
```mermaid
sequenceDiagram
  actor 사용자 as 사용자
  participant API as API
  participant 주문 서비스 as 주문 서비스
  participant DB as DB
  participant 외부 데이터 플랫폼 as 외부 데이터 플랫폼

activate 사용자
  사용자 ->>+ API: 상품 주문
  API ->>+ 주문 서비스: 상품 주문 처리
  주문 서비스 ->>+ DB: 상품 재고 조회
  activate DB
  DB ->> 주문 서비스: 상품 재고 반환
  deactivate DB
  opt 상품 재고 없음
    주문 서비스 -->> API: 상품 주문 실패
  end
  주문 서비스 ->> 주문 서비스: 주문 금액 계산
  opt 쿠폰 사용
    주문 서비스 ->> DB: 쿠폰 조회
    activate DB
    DB -->> 주문 서비스: 쿠폰 조회 결과
    deactivate DB
    alt 유효한 쿠폰
    주문 서비스 ->> 주문 서비스: 쿠폰 적용 주문 금액 계산
    else 유효하지 않은폰
      주문 서비스 -->> API: 상품 주문 실패
    end
  end
  주문 서비스 ->> DB: 잔액 조회
  activate DB
  DB -->> 주문 서비스: 잔액 조회 결과
  deactivate DB
  opt 잔액 < 주문 금액
    주문 서비스 -->> API: 상품 주문 실패
  end
  주문 서비스 ->> DB: 상품 재고 감소
  activate DB
  DB -->> 주문 서비스: 상품 재고 감소 결과
  deactivate DB
  opt 쿠폰 사용
    주문 서비스 ->> DB: 쿠폰 감소(사용)
    activate DB
    DB -->> 주문 서비스: 쿠폰 감소(사용) 결과
    deactivate DB
  end
  주문 서비스 ->> DB: 상품 주문 저장
  DB -->> 주문 서비스: 상품 주문 저장 결과
  주문 서비스 ->> DB: 잔액 감소
  activate DB
  DB -->> 주문 서비스: 잔액 감소 결과
  deactivate DB
  주문 서비스 -) 외부 데이터 플랫폼: 주문 정보 전송
  주문 서비스 -->>- API: 상품 주문 성공
  API -->>- 사용자: 결과 전달
deactivate 사용자
```

## 상위 상품 집계
```mermaid
sequenceDiagram
        participant 스케줄러
        participant 주문
        participant DB

loop 주문 통계
opt 현재 시간 == 00시00분00초
    스케줄러 ->> 주문: 작업 요청
    주문 ->> 주문: 주문 정보 집계
    주문 ->> DB: 통계 데이터 저장
    DB -->> 주문: 통계 데이터 저장 결과
    주문 -->> 스케줄러: 작업 완료
end
end
```
